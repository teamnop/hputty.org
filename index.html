HPuTTY

PuTTY is a free implementation of Telnet and SSH for Win32 and Unix platforms, along with an xterm terminal emulator.

PuTTYTray + iPuTTY + @ = HPuTTY ( 한글 PuTTY )

기능


HPuTTY

  • Unicode fallback font 지원

iPuTTY에서 가져온 기능

  • On-the-spot IME 지원
  • CP949/UTF-8 빠른 변경 메뉴
  • ANSI/유니코드 폰트 분리
  • 투명도 조절 단축키
  • Ctrl+Tab키로 다른 putty 창으로 전환
  • 윈도우 버전을 인식하여 폰트 지정
  • UTF-8을 기본 인코딩으로 설정
  • Italic체 지원

PuTTYTray에서 가져온 기능

  • 시스템 트레이에 최소화 기능
  • 커스터마이즈 가능한 아이콘
  • bell signal을 받았을때 창 깜빡임
  • 창 투명도 설정
  • URL 하이퍼 링크
  • 설정을 파일로 저장
  • 빠른 창 고정 기능
  • 접속 실패시 자동 재접속 지원
  • Android adb 지원

소스코드는 위의 GitHuB 저장소 에서 자유롭게 받으실 수 있습니다.
